# Sample config (repo)

Committed examples copied into **`.local/config/`** on first run when those files are missing:

| File | Becomes |
|------|---------|
| [secrets.json](secrets.json) | `.local/config/secrets.json` (placeholder email + app-password shape) |
| [card_rules.json](card_rules.json) | `.local/config/card_rules.json` (merged with placeholder `card` + `passwords`) |
| [tags.json](tags.json) | `.local/config/tags.json` |

Edit the generated local files; do not commit real secrets or personal rules.
